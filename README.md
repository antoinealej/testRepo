# Lunar Exploration History

This project is a fully static, accessible, and responsive React and TypeScript webpage presenting the history of lunar exploration. It includes structured content, images, and navigation to provide an engaging overview of major missions, milestones, and future initiatives related to the Moon.

## Development

- Run `npm install` to install dependencies.
- Run `npm run start` to start the development server.
- Run `npm run build` to build the production bundle.
- Run `npm run lint` to check code quality.

## Technologies

- React 18
- TypeScript
- Vite
- ESLint
- Prettier

## Accessibility

The webpage is designed with accessibility in mind, including semantic HTML, alt text for images, and sufficient color contrast.
