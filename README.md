# Lunar Exploration History

This project is a static React-based webpage presenting the history of lunar exploration. It includes a timeline of major missions, milestones, images, and future initiatives.

## Development

- Install dependencies: `npm install`
- Run development server: `npm run start`
- Build for production: `npm run build`
- Preview production build: `npm run preview`

## Features

- Fully static webpage with no backend
- Responsive design for desktop, tablet, and mobile
- Accessible typography and color contrast
- Descriptive alt text for all images
- Optimized performance with compressed images and minimal scripts

## Browser Support

Tested on Chrome, Firefox, Safari, and Edge (latest versions).
